# Security Persona Guide

## Overview

The Security persona provides a security-focused view of the Rule Repository, centered on security policy enforcement, vulnerability tracking, and access control compliance.

## Getting Started

1. Navigate to the Security dashboard at `/security`
2. Your default view shows the Security Overview Dashboard
3. Use the sidebar to access security policies, alerts, and audit information

## Key Workflows

### Security Policy Monitoring
- Monitor compliance with security policies across the organization
- Review security-related rule evaluations and violations

### Vulnerability and Risk Tracking
- Track security findings and their remediation status
- Assess risk levels for reported issues

## Vocabulary
- **Rule** = Security policy, access control requirement, or vulnerability standard
- **Violation** = Configuration, code, or practice that fails to meet a security policy
- **Evaluation** = Security assessment against applicable security rules
- **Subject** = Code change, configuration, or system event under security review
